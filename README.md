# EEG Signal Classification for Epileptic Seizure Detection

## Overview
This project focuses on classifying EEG (Electroencephalogram) signals to detect epileptic seizure activity using machine learning techniques. The goal is to analyze brain signal patterns, preprocess EEG data, train classification models, and evaluate their ability to distinguish seizure-related activity from normal brain activity.

The project uses the Epileptic Seizure Recognition dataset from Kaggle and demonstrates a complete machine learning workflow including:
- Data exploration and preprocessing
- Signal analysis and feature preparation
- Model training and evaluation
- Visualization of EEG patterns and classification performance

Dataset:
https://www.kaggle.com/datasets/harunshimanto/epileptic-seizure-recognition
